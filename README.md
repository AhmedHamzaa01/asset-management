# Asset Management API

## Overview
A REST API for managing security assets (domains, subdomains, IPs, services, certificates, technologies) with deduplication, lifecycle tracking, and relationship graph support.

## Tech Stack
- **Python 3.11+**
- **FastAPI** - Web framework
- **PostgreSQL** - Database
- **SQLAlchemy** - ORM
- **Docker** - Containerization
- **JWT** - Authentication

## Quick Start

### Prerequisites
- Docker Desktop
- Python 3.11+
- Git
